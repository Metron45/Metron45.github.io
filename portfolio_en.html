Language: C++
Project made on "Projektowanie Efektywnych Algorytmów" course. Its purpose being writng a solution to TSP problem using TSP-lib example problems. Currently it contains solution algorithsms such as Bruteforce, Dynamic Programming, Simmulated Annealing and Genetic Algorithm.
